Elsa - Voice Controlled Assistant

Elsa is a Python-based voice-controlled assistant that assists users with various tasks using speech recognition and text-to-speech conversion. This project aims to provide users with a hands-free experience for tasks such as web browsing, music playback, system control, and more.

Features:

- Speech Recognition
- Text-to-Speech Conversion
- Web Browsing
- System Control
- Music Playback
- Camera Access

Usage:

1. Clone this repository to your local machine.
2. Make sure you have Python 3.x installed.
3. Install the required dependencies using `pip install -r requirements.txt`.
4. Run `main.py` to activate Elsa.
5. Wait for the activation prompt, then speak your command.

Example commands:

- "Elsa, open Google"
- "Elsa, play some music"
- "Elsa, what's the time?"
- "Elsa, shut down the system"
